# Manmeet Virdi — Personal Portfolio

A personal 3D portfolio website built with React, TypeScript, Three.js, and GSAP animations.

**Live site:** https://manmeet-virdi-portfolio.vercel.app

## About

Hi, I'm Manmeet Virdi — a Full Stack & Mobile App Developer based in Vancouver, BC. This portfolio showcases my projects and experience built during my Post-Degree Diploma in Web & Mobile App Design and Development at Langara College.

## Projects Featured

- **Semina** — AI Plant Care Mobile App (Capstone) — [seminaca.vercel.app](https://seminaca.vercel.app)
- **BloomUp** — Parenting Support Platform (Team Project) — [bloomup.wmdd.ca](https://bloomup.wmdd.ca)
- **ParkSmart** — Vancouver Parking Finder (Team Project) — [parksmart-sable.vercel.app](https://parksmart-sable.vercel.app)

## Tech Stack

- React 18 + TypeScript + Vite
- Three.js + React Three Fiber
- GSAP animations
- CSS Modules

## Getting Started

```bash
git clone https://github.com/ManmeetVirdi17/Manmeet-Virdi-Portfolio.git
cd Manmeet-Virdi-Portfolio
npm install
npm run dev
```
